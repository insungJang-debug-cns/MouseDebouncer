# Mouse Debouncer

마우스 버튼 채터링(이중 클릭) 방지 유틸리티. 설정한 딜레이(ms) 이내의 재입력을 차단합니다.

## 설치 및 실행

별도 설치 불필요. `MouseDebouncer.exe`를 실행하면 시스템 트레이에 상주합니다.

> .NET 8 런타임이 없는 PC에 배포할 때는 단일 exe 퍼블리시 빌드를 사용하세요.  
> (`dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true`)

## 기능

### 설정 창

트레이 아이콘 더블클릭 또는 우클릭 → **Open Settings** 로 열 수 있습니다.

| 항목 | 설명 |
|---|---|
| XButton1 Delay (ms) | 사이드 버튼 1 딜레이 (기본값: 300ms) |
| XButton2 Delay (ms) | 사이드 버튼 2 딜레이 (기본값: 300ms) |
| Left Delay (ms) | 왼쪽 버튼 딜레이 (기본값: 0, 비활성) |
| Right Delay (ms) | 오른쪽 버튼 딜레이 (기본값: 0, 비활성) |
| Middle Delay (ms) | 휠 클릭 딜레이 (기본값: 0, 비활성) |
| 시작 프로그램 등록 | Windows 시작 시 자동 실행 여부 |
| 누른 버튼 | 현재 창이 열려 있을 때 마우스 버튼을 누르면 어떤 버튼인지 표시 |
| Blocked | 오늘 차단된 클릭 수 (자정 리셋) |

- 딜레이를 `0`으로 설정하면 해당 버튼 디바운싱 비활성화
- 단위: ms (100 = 0.1초, 1000 = 1초)
- **Save** 버튼을 눌러야 설정이 저장됩니다

#### 내 마우스 버튼 확인 방법

설정 창을 열어둔 상태에서 확인하고 싶은 버튼을 누르면 **"누른 버튼:"** 항목에 버튼 이름이 표시됩니다.  
해당 이름의 Delay 항목에서 딜레이를 조정하세요.

### 트레이 아이콘

| 메뉴 | 설명 |
|---|---|
| Open Settings | 설정 창 열기 |
| Disable / Enable | 디바운싱 일시 중지 / 재개 |
| Blocked today | 오늘 차단된 클릭 수 |
| Exit | 프로그램 종료 |

- 설정 창의 X 버튼은 종료가 아닌 트레이로 최소화합니다
- 비활성 상태일 때 트레이 아이콘이 방패 → 경고 아이콘으로 변경됩니다

## 설정 파일

`config.json` — 실행 파일과 같은 폴더에 자동 생성됩니다.

```json
{
  "Left": 0,
  "Right": 0,
  "Middle": 0,
  "XButton1": 300,
  "XButton2": 300
}
```
